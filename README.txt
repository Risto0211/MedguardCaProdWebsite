To meet the requirements, you may install following packages in advance (also written in the package.json):
-npm
-node.js
-vue3
-vue cli
-express
-jsonwebtoken
-bcryptjs
-body-parser
-cors
-sqlite3
-vue-router
-vue-template-compiler


To run the code, first loacte to the root directory and run the server.js in one terminal using:
node src\server.js

Then in another terminal open the webpage in the browser using:
npm run serve

The frontpage can be either open in localhost or a default ip address
